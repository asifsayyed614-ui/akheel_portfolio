# KK Mohammed Akheel — Portfolio (React + Vite)

## Setup
```
npm install
npm run dev      # local dev server
npm run build    # production build -> dist/
```

## Notes
- Add your profile photo as `public/photo.png` (referenced in the Hero section as `/photo.png`).
- Built with React 18, Vite, and Tailwind CSS.
- The animated glowing-particle background lives in `src/components/BackgroundCanvas.jsx`.
- Tailwind's CDN script and the in-browser Babel transform used in the original single-file HTML have been replaced with a proper Vite build pipeline — same visuals, production-ready setup.
